Desarrollos

Freetech Sender: desarrollo de sistemas de generación automático de llamadas, integrado a Asterisk / Elastix. Usamos Python/Django para el frontend web, daemons Python, Celery y Redis para el backend, automatizamos setup de servidores y deploy, usamos contenedores Docker para tener un ambiente de desarrollo muy similar al productivo.
